# Forge Party Inventory - Firefox Addon
This add-on allows you to download the party inventory as a csv at will. 

It is currently available at the following url: https://addons.mozilla.org/en-US/firefox/addon/forge-vtt-party-inventory/

Absent that, navigate to the latest release, download the zip file, extract it and follow the steps at the site below for adding it to your firefox instance manually.

[Temporary Firefox Extension](https://extensionworkshop.com/documentation/develop/temporary-installation-in-firefox/)

Preferably wait for the release mentioned above.
